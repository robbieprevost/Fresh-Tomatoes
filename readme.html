FRESH TOMATOES:

Fresh Tomatoes is a simple app that presents a website containing a list of featured movies and allows users to view the movie's poster and trailer from YouTube.

Along with providing featured movies, the app provides acces to Wikipedia's comprehensive lists of thousands of films.

The app parses through Wikipedia pages and search results to provide access to a movie's poster image and YouTube trailer.

Getting started:

  • Unzip the files from the downloaded folder
  • Go the the location of the files in a command prompt and type: python hub.py
  • The app should start and will create a some small json files from the results it gathers as it downloads and parses through the lists of films from Wikipedia
  • After 10-20 seconds, go to 127.0.0.1:8080 in your browser and enjoy
